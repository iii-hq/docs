The iii SDK is the base SDK for building iii applications. It provides the core functionality for building iii applications.

## Install the SDK

<Tabs items={['Node', 'Python']} groupId="language" persist>
  <Tab value="Node">
    <DynamicCodeBlock lang="bash" code="npm install @iii-dev/sdk" />
  </Tab>
  <Tab value="Python">
    <DynamicCodeBlock
      lang="bash"
      code={`uv pip install iii-sdk
# or
pip install iii-sdk`}
    />
  </Tab>
</Tabs>

## Usage

Run `iii` command line, this will start the iii process.

<DynamicCodeBlock lang="bash" code="iii" />

<Tabs items={['Node', 'Python']} groupId="language" persist>
  <Tab value="Node">
    <DynamicCodeBlock lang="typescript" code={`import { Bridge } from '@iii-dev/sdk'

const bridge = new Bridge(process.env.III_BRIDGE_URL ?? 'ws://localhost:49134')`} />

  </Tab>
  <Tab value="Python">
    <DynamicCodeBlock lang="python" code={`import asyncio
import os
from iii import Bridge

async def main() -> None:
bridge = Bridge(os.getenv("III_BRIDGE_URL", "ws://localhost:49134"))

    @bridge.function("greet.hello")
    async def hello(data):
        name = data.get("name", "World")
        return {"message": f"Hello, {name}!"}

    await bridge.connect()
    await asyncio.Event().wait()

if **name** == "**main**":
asyncio.run(main())`} />

  </Tab>
</Tabs>

<Tip>The port to iii is usually `ws://localhost:49134` if you're running the iii process locally.</Tip>
