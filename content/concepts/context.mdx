---
title: 'Context'
---

Context is the full state of a iii application and is what enables application code to interact with functionality available to the iii Engine such as state, databases, queues, Triggers, and logging.

<Callout title="Side Effects" type="info">
  The Context object makes it possible for any Function to trigger an invocation of any other Function within the iii
  Engine.
</Callout>
