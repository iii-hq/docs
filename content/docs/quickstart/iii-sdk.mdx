## Description

iii SDK is the base SDK for building iii applications. It provides the core functionality for building iii applications.

## Installation

```bash
npm install @iii-dev/sdk
```

## Usage

Run `iii` command line, this will start the iii process.

```bash
iii
```

```typescript
import { Bridge } from '@iii-dev/sdk';

const bridge = new Bridge(process.env.III_BRIDGE_URL ?? 'ws://localhost:49134')
```

<Tip>
  The port to iii is usually `ws://localhost:49134` if you're running the iii process locally.
</Tip>
